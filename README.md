** Cybersecurity Lab Writeups**

A collection of penetration testing and threat simulation reports from university-based lab environments. All content is original, sanitised, and intended for professional showcase only.

**Contents**

- **Internal Network Pentest:** Multi-host enumeration and exploitation using Nmap, Gobuster, and Metasploit.
- **Industrial Environment Simulation:** OT/ICS-focused test on PLCs and Modbus systems, aligned with IEC 62443.
- **Group Pentest Project:** Team-based network recon, SMB exploitation, and root access simulation.


Disclaimer: All environments were legally controlled lab setups for academic learning. No part of this repository is intended for reuse in assessments or commercial platforms.
